# Land Resource Thesis Knowledge Feed

Public, machine-readable update feed for the land-resource thesis assistant.

- `manifest.json` identifies the latest signed package.
- `packages/` contains exact package bytes verified by SHA-256 and Ed25519.
- Every document is a pending candidate and requires manual review in the desktop application.
- The feed contains public-source indexes and academic metadata only. It does not contain user data, API keys, private papers, or paywalled full text.
